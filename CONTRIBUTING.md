# Contributing to Aliensit

Thank you for considering contributing to the Aliensit project! Here are some
guidelines to help you get started.

## How to Contribute

1. **Fork the repository**: Click on the fork button at the top right of the
   project page.
1. **Clone your fork**: Use the command `git clone <your-fork-url>` to clone the
   repository to your local machine.
1. **Create a branch**: It's good practice to create a new branch for each
   feature or bugfix you work on. Use the command
   `git checkout -b <branch-name>`.
1. **Make your changes**: Edit the files as needed to implement your changes.
1. **Commit your changes**: Use `git commit -m "Your commit message"` to save
   your changes.
1. **Push your changes**: Push your branch back to your fork using
   `git push origin <branch-name>`.
1. **Open a pull request**: Navigate to the original repository, select your
   fork and branch, and click on the "New Pull Request" button.

## Code of Conduct

Please adhere to the [Code of Conduct](CODE_OF_CONDUCT.md) in all interactions
with the community.
